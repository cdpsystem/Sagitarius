Introduction

Angular Project Start it's a bundle composed by Angular 6 and some of the most used packages in Angular. Using the router system and the router outlets provided by Angular 6, this bundle come with all the requeriments solved and a first code writed, just for download and start your application, without needs to install the usual packages.

Installed Packages

This packages comes with the bundle

Bootstrap.css

Bootstrap its a complete css suite for develop full responsive sites without need overhead with the responsiveness

Popper.js

Popper.js Allows you to create powerfull drop downs. It's a dependency of Bootstrap, but you can use it along

JQuery

One of the most famous and simplest frameworks for javascript.

Moment.js

Moment js its a library for javascript for a better time and date control.

Sass Support

Syntactically Awesome Style Sheets (Sass) and his new syntax SCSS are bundled in this pack for a better css syntax, totally configured with Angular filesystem

Chart.js

Another library. This is used to create simple and beautiful graphics

ngx loading bar

A little tool for display a loading bar. Configured for use it with the Angular router's technology
